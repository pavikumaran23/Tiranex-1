📌 Overview

This project is a Sales & Revenue Analysis Dashboard built using Microsoft Power BI. It helps analyze business performance through interactive visuals, KPIs, and filters.

The dashboard provides insights into:

Total Revenue
Sales Quantity
Product Distribution
Regional Performance
Time-based Trends
🎯 Objectives
Analyze overall sales and revenue performance
Identify top-performing products and categories
Compare revenue across regions
Track trends over time (daily/quarterly)
Enable interactive filtering for better insights
📂 Dataset

The dataset used contains the following fields:

Date
Product
Category
Region
Quantity
Price
Revenue
